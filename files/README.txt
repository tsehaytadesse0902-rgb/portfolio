CV File Instructions:

1. Place your CV file in this folder
2. Name it: Tsehaye_Tadesse_CV.pdf
3. The download button will automatically work once the file is added

Supported formats:
- PDF (recommended): Tsehaye_Tadesse_CV.pdf
- Word Document: Tsehaye_Tadesse_CV.docx

Note: If you use a different filename or format, update the cvPath variable in js/script.js:
- For PDF: 'files/YourFileName.pdf'
- For Word: 'files/YourFileName.docx'

The download functionality is now active and ready to use!
